Javascript loops and iterations

posted on Nov. 16, 2020 by Navneet

Do you face difficulties while using loops in JavaScript? If yes, then you are at the right place. After reading this article, you will understand what each of those loops are doing.

Let’s start with understanding the meaning of a loop in JavaScript. Loop is a special command that allows us to do something repeatedly. Everytime the code in the loop is executed, it is called the iteration of a loop. A loop like a GIF image, do the same thing repeatedly.

There are five different types of loops in JavaScript:

  • While loop
  • Do while loop
  • For loop
  • For of loop
  • For in loop

While loop

Syntax-

            while(condition) {
              Statements
            }

In the “while” loop, a set of statements executes until a specific condition is true. Before executing the code, a condition is checked. If the test condition evaluates to true, code is executed. A very light example is eating until you are full. Before having a bite, check if you are not full. Similarly, before executing the code, check if the condition is true.

Example

            const i = 0;
            while ( i > 0){
              console.log(‘value of i is’ + i);
            }

            output-

Do while loop

Syntax-

          	do{
          	  statements
          	} while (condition);

In the “do while” loop, the computer executes the code once at least before the condition is checked. If the condition evaluates to true, code gets executed again or else the loop will stop.

Example-

          	const i = 0;
            	do {
            	  console.log(‘value of i is ‘ + i );
              } while ( i < 5);  
              
            output- 
              value of i is 0
              . 
              . 
              value of i is 5

In the above example, computer will log ‘value of i is 0’. Then it will check the condition ‘i > 0’ which will evaluate to false and loop will stop. Hence, the set of statements will execute at least once.

For loop

Syntax-

            	for([initialize expression]; [condition]; [increment expression]) {
            	  statements
            	}

In the “for” loop, you can initialize an expression ( let i = 0 ) and in every iteration, you can update that expression . You can skip the initialization of an expression and just put a semi-colon instead. Like “while” loop, a “for” loop is also executed until the condition is true.

Example-

          	for(let i = 0; i < 10; i++){
          	  console.log( i + ‘ is less than 10.’);
            }    
            
            output-
              0 is less than 10.
              .
              .
              .
              9 is less than 10.

In the above example, a variable “i” is initialized, and is incremented in every iteration. A condition is checked that if “i” is less than 10, then execute the code.

For of loop

Syntax-

          	for(let item of array){
          	  statements
          	}

The “for of loop” is a special loop that iterates over an array of items and executes the set of statements for every particular item.

Example-

          	const basket = [‘apple’, ‘banana’, ‘mango’, ‘kiwi’];
       	    for(let fruit of basket) {
          	  console.log(fruit);
             }  
             
            output-
              apple
              banana
              mango
              kiwi

In the above example, there is an array of fruits called basket. “For of loop” is iterating over all the items of the array and logging them into the console.

For in loop

Syntax-

          	for(let item in object){
          	  statements
          	}

The “for in” loop is a special loop, which iterates over an object and execute the set of statements for that particular item.

Example-

          	const cart = {
          	  Juice: 12,
              Cake: 1,
          	  Apples : 1,
          	};
            for(let items in cart){
          	  console.log(item);
            }   
            
            output-
              12
              1
              1

In the above example, there is an object named cart. The loop is going over all the items in cart object one by one, and for every item, the loop executes the code.
